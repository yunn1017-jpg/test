# 📝 To-Do List 관리 프로그램

콘솔 기반의 간단한 할 일 관리 프로그램입니다.

## 기능

- **할 일 보기** — 등록된 할 일 목록을 조회합니다.
- **할 일 추가** — 새로운 할 일을 추가합니다.
- **할 일 완료** — 완료한 항목을 체크합니다.
- **할 일 삭제** — 필요 없는 항목을 삭제합니다.
- 데이터는 `todos.json` 파일에 자동 저장됩니다.

## 실행 방법

```bash
python test.py
```

## 사용 예시

```
========================================
  📝 To-Do List 관리 프로그램
========================================

[1] 할 일 보기  [2] 추가  [3] 완료  [4] 삭제  [0] 종료
선택: 2
추가할 할 일을 입력하세요: 파이썬 공부하기
✅ '파이썬 공부하기' 추가 완료!
```

## 요구 사항

- Python 3.6 이상
